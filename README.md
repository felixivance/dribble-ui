# Dribble UI

1. NextJs front end application to simulate the Dribble UI

- Web version only
- Similar to almost alike font and images
- Tests to test the components

## INSTALLATION

To run this project run

```bash
  yarn run dev
```

## Demo

Visit http://localhost:3000
